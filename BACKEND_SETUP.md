# Backend Kurulum Rehberi

Backend sunucusu bulunamadı. Backend'i kurmak için aşağıdaki adımları izleyin:

## Seçenek 1: Backend'i Başka Bir Yerde Başlatın

Eğer backend projeniz başka bir klasördeyse:
1. Backend klasörüne gidin
2. `npm install` çalıştırın
3. `.env` dosyasını kontrol edin
4. `npm run dev` ile başlatın

## Seçenek 2: Backend Oluşturma

Backend projesi yoksa, backend'i oluşturmamız gerekiyor. Bu işlem için:
- Node.js/Express backend
- Prisma ORM
- JWT authentication
- PostgreSQL/MySQL veritabanı

gerekli.
